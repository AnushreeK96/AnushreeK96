Steps to open the project by Executable Code -

1. Navigate to Folder - \Executable Code\Covid-Statistics-Viewer
2. Open the index.html file in the browser

Steps to run the local host by Source Code - 

1. Navigate to \Source Code and open a terminal
2. Run the command 'npm install'
3. Run the command 'ng serve'
4. View the running application as 'http://localhost:4200/' - on the browser
